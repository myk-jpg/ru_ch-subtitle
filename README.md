# 项目内容

本仓库提供ASS格式俄语原文字幕和中文字幕  
.ru 文件为俄语原文字幕  
.ch 文件为中文翻译字幕  
.ru_ch 文件为俄语中文双字幕  
如有内容与翻译错误或时轴不对应，欢迎修改  

## 📥 使用方法
1. 从官方渠道获取原视频
2. 下载对应的 `.ass` 字幕
3. 将字幕与视频放在同一文件夹
4. 文件名一致即可自动加载，也可手动拖入视频加载  

本仓库提供视频的下载链接，转载开源以获得原作者许可  
如需下载视频资源可以在仓库的 LINK.txt 文件中复制下载地址并下载
