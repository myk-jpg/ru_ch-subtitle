# ru_ch-subtitle
ANSYS CFX 教学视频的字幕文件

请支持原作者：Олег Батурин教授
原视频作者频道：https://www.youtube.com/@ОлегБатурин-п8к

# 俄语、中文 ASS 字幕（非官方）

本仓库提供 **ASS 格式中文翻译字幕**  
字幕全部由MYKC个人制作并提供开源下载
仅供学习与交流使用。

## 📥 使用方法
1. 从官方渠道获取原视频
2. 下载对应的 `.ass` 字幕
3. 将字幕与视频放在同一文件夹
4. 文件名一致即可自动加载，也可手动拖入视频加载

## 🅰 字体说明
字幕使用：
- Times New Roman
- 宋体（SimSun）

建议使用PotPlayer播放

Windows 用户一般无需额外安装  
macOS / Linux 用户可能会看到字体替换
